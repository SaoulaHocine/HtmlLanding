About Us

FileFlow is a focused, three-step interface for organizing files into folders. It currently runs as a safe dry‑run in the browser so you can validate your setup before any real file operations, with an Electron mode planned for full system access.

Built with modern web tech — Next.js (App Router), React 18, TypeScript, Tailwind CSS, and shadcn/ui — it leverages the File System Access API in Chromium-based browsers for seamless folder selection.

  • Clear Copy/Move selector right at the start
  • Smart grouping by category, date, extension, size, A–Z, or keywords
  • Destination subfolders with conflict handling (rename, replace, skip)
  • Dry‑run flow with a comprehensive summary

Next up: preferences persistence, preview of resulting folder names, and optional Electron integration for real copy/move with progress and safety prompts.
